What is GOmsg?

GOmsg is a website where people can talk about their interests. It is inspired by chan style boards and other message forums. There is a limit amount of threads per board and each thread has a limit amount of replies. Once the limit amount of replies is reached, the thread dies. If the limit amount of threads have already been created and another thread wants to be created, the oldest thread is killed so that the new one can be borne.

This project is written in GO and it is a way for me to practice the implementation of the modules net/http and text/template so that I can use these modules for bigger projects that I have planned. This website use no javascript and it is a sort of proof of work that websites do not need to be bloated as most of the modern web is. This project will also help me work on my CSS skills.
